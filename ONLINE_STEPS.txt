CAREER CROX ONLINE STEPS

1. Create a NEW Supabase project.
   - Project name: Career-Crox-Interns
   - Enable Data API: ON
   - Automatic RLS: OFF

2. From Supabase dashboard copy the NEW working Project URL.
   - It must open in browser.
   - Replace SUPABASE_URL in .env with that new URL.

3. In Supabase open SQL Editor.
   - Open docs/SUPABASE_SCHEMA.sql from this project.
   - Paste all SQL into SQL Editor.
   - Run it.

4. Open terminal in this project folder and run:
   py tools\migrate_sqlite_to_supabase.py

5. Then run the app:
   py app.py

6. Open browser:
   http://127.0.0.1:5000

7. If everything looks correct, upload this project to GitHub.
   IMPORTANT: do NOT upload the .env file.

8. Create a Render Web Service from the GitHub repo.
   Set these environment variables in Render:
   - SECRET_KEY
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_ROLE_KEY

9. Build / Start commands on Render:
   Build: pip install -r requirements.txt
   Start: gunicorn app:app

10. After Render deploy, all new edits/additions will go to Supabase.
