Career Crox Interns CRM - quick local test

1) Open terminal in this folder
2) Run: py -m pip install -r requirements.txt
3) Run: py tools\import_excel_to_supabase.py
4) Run: py app.py
5) Open: http://127.0.0.1:5000

This package already includes a .env file for the current testing Supabase project.
Do not upload the .env file to public GitHub.
